鉴于有人还不会些加密的方法（无针对），这里写几个

base64

可以解密，没有安全性。

import base64
r = "yourpassword".encode("utf-8")
s = base64.b64encode(r)
print(s)

hash

不能解密，但是可以验证。也就导致可以被彩虹表破解。

from hashlib import sha256
r = "yourpassword".encode("utf-8")
s = sha256()    # Get the hash algorithm.
for i in range(1000000): # prevent rainbow table.
    s.update(r)    # Hash the data.
b = s.hexdigest()       # Get he hash value.
print(b)

AES

对称加密。

RSA

非对称加密。可以根据密钥解密，也可以验证。 用法：

原文+公钥=a
公布a，公钥
对方由密钥得到原文

以上都是常用的加密方法，接下来要介绍的是我想到的不能解密可以验证的加密方法。

设密码为k，对其进行100000次（多次）哈希加密后记为a
令f(k)=大于a的最小质数
生成另一大质数p
f(正确密码)×p=c，存c
输入密码k，f(k)整除c大致等价于密码正确

我的网站 就用了这个算法。
